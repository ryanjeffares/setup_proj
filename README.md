# setup_proj

A Python script that will spit out the skeleton of a new basic C++ CMake project.

Install it system wide with:

```
sudo cp setup_proj.py /usr/bin/setup_proj
sudo chmod +x /usr/bin/setup_proj
```

Usage:
```
setup_proj <project-name>
```
